# 🦊 Must-b
**Your AI Brain | Cloud Brain, Local Muscle.**

Must-b is an advanced, autonomous AI agent platform designed to execute complex workflows, manage dynamic plugins, and provide real-time environmental context. Built on the Auto Step Platform architecture, Must-b bridges the gap between cloud intelligence and local execution.

## 🌟 Core Features
* **🛡️ The War Room:** A cinematic, zero-artifact interface for centralized control of all AI activities and parallel processing slots.
* **🧩 Plugin Architect:** An autonomous module that allows Must-b to write, test, and deploy its own skills dynamically.
* **🧠 Semantic Memory:** Integrated local Vector DB for long-term memory retrieval and context awareness.
* **👁️ Live Sight:** Real-time visual and environmental data processing.
* **⚡ Cross-Platform:** Seamlessly integrated across Web, macOS, iOS, and Android.

## 🚀 Getting Started
```bash
npm install
npm run dev
```
