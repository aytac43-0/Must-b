# `must-b setup`

Initialize `~/.must-b/must-b.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
must-b setup
must-b setup --workspace ~/.must-b/workspace
```

To run the wizard via setup:

```bash
must-b setup --wizard
```
